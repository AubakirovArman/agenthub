# AgentHub Fixtures

Small fixture projects used by PRD v3 product smoke scripts.

- `rust-basic`: minimal Rust crate.
- `python-data`: data workspace artifact.
- `terraform-basic`: infra workspace artifact without running `terraform apply`.
- `content-basic`: content workspace artifact.

The reference web fixture lives in `examples/reference-web-app` because it is also used by integration tests.
