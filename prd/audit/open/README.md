# Open PRD Areas

These are explicit PRD items that are not fully implemented yet.

- Specialized `backend_tdd` verifier profile.
- Specialized `db_migration` verifier profile.
- Command policy enforcement for `safe`, `needs_approval`, and `restricted` command lists.
- Sandbox Level 1, Level 2, and Level 3 execution isolation beyond current local controlled process supervision.
- Real remote runner execution.
- Networked central policy server; current implementation supports file-backed central policy via `AGENTHUB_POLICY_PATH`.
- Cryptographic plugin package signature verification; current implementation stores optional signature metadata.
- Metrics dashboards and KPI aggregation.
- Full WAL subsystem.
- Full end-to-end reference web application fixture for "Add Page to Existing Web App".
